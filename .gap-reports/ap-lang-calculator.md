# ap-lang-calculator
verdict: already-wins

## Competitors
1. appass.com (AP Pass) — MC count + three FRQ score inputs, weighted composite (MC 45% / FRQ 55%), maps to predicted 1–5. Same methodology we use; UI is plain number entry.
2. collegevine.com AP Lang calculator — MC + essay sliders, live predicted 1–5, plus marketing/context blurbs. Functionally identical inputs/outputs.
3. savemyexams / apscorecalculator style tools — MC + 3 essays, composite, predicted score, sometimes a note that curves change yearly.

## Our current features
- MC (of 45) + three FRQ inputs (synthesis, rhetorical analysis, argument, 0–6 each).
- Weighted composite out of 150 (MC 45% / FRQ 55%), live predicted 1–5 score.
- Full score-band scale (5→1 with composite ranges) with the active band highlighted.
- Input clamping to valid ranges. Honest "College Board re-curves yearly, estimate only" disclaimer. Fully client-side.

## Gaps to close (only genuine missing/worse features; empty if already-wins)
(none — we match the standard methodology and additionally show the full cut-point scale with the active band highlighted, which most competitors do not. Slider vs number entry is cosmetic.)
