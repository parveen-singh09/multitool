# margin-calculator
verdict: already-wins

## Competitors
1. calculator.net/margin-calculator — key features: enter any two of cost/revenue/margin/profit to derive the rest; margin %, profit, markup %; cost/profit breakdown chart; also bundles stock-trading and currency-exchange margin calculators; save.
2. omnicalculator.com/finance/margin — key features: any-two-solve-the-rest (cost/margin/revenue/profit); markup + margin; links to VAT/sales-tax/discount variants; Excel guidance.
3. shopify.com profit margin calculator — key features: cost + markup → revenue, profit, gross margin; single-direction, marketing-oriented.

## Our current features
- Cost + sale price → profit, gross margin %, markup %.
- Reverse panel: cost + target margin → required sale price.
- Fully client-side.
